For this project, me and my team adapted the common A* pathing algorithm to a simple maze game, to create an enemy AI. I personally made the entire game framework using python, and integrated the algorithm, which I developed with my partner, into the game's systems. Synthesizer Project

In this project, me and my group of 4 made this fully functioning virtual synthesizer. I personally was responsible for the audio processing, which included creating a stereo delay, implementing an external reverb fucntionality, and creating the basic oscillators for the synthesizer, including a Low Frequency Oscillator(LFO). It is a fully playable monophonic instrument, and has full MIDI implementation, allowing an external keyboard to be used to control it.
